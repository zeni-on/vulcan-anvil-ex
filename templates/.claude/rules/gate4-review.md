# Gate 4: QA 리뷰 규칙

## 3단계 분리 투입

Gate 4는 3단계로 구성된다. 각 단계를 별도 서브에이전트로 투입한다.

1. **코드 리뷰** — 요구사항 + 설계 + 소스코드 기반, 리뷰 메모 산출
2. **테스트 실행** — Test-Plan.md + ENVIRONMENT.md 기반, 테스트 결과 + 스크린샷 산출
3. **최종 판정** — 1단계 + 2단계 결과 종합, `docs/04-review/REQ-NNN-Review.md` 산출

## 규칙

- 🔴 필수 수정 발견 시 Developer 재투입 → 재검증
- 각 단계 내에서도 REQ 청크 분할을 적용한다
