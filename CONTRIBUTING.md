# 기여 안내

Vulcan-Anvil Ex는 이슈, 아이디어, 문서 개선, Pull Request를 환영합니다.

이 프로젝트는 Apache License 2.0으로 공개됩니다. 기여를 제출하면 해당 기여도 이 프로젝트와 같은 라이선스 조건으로 제공하는 것에 동의한 것으로 봅니다.

## 기여 전 확인 사항

기여를 제출하기 전에 다음을 확인해 주세요.

- 본 기여를 제출할 권한이 있습니다.
- 회사 기밀, 비공개 정보, 고객별 정보, 회사 내부 정보가 포함되어 있지 않습니다.
- 비공개 프로젝트 코드, 비공개 프로젝트 문서, 내부 표준, 보안 규칙, 로그, 인증 정보, 고객 데이터를 복사하지 않았습니다.
- 고용 관계 또는 위탁 업무 중 작성한 기여라면, 소속 조직 또는 고객이 공개 프로젝트 제출을 허용했습니다.
- 포함한 제3자 코드, 템플릿, 문서는 이 프로젝트 라이선스와 호환되며, 필요한 경우 출처를 명확히 표시했습니다.

## Pull Request 기준

Pull Request를 보낼 때는 다음 기준을 지켜 주세요.

- 변경 범위를 작고 명확하게 유지합니다.
- 왜 필요한 변경인지 설명합니다.
- 가능하면 검증 명령을 함께 남깁니다.
- 생성 캐시, 빌드 산출물, 로컬 경로, 비밀값, 샘플 프로젝트 데이터를 포함하지 않습니다.
- `docs/ref-docs/`, `.vulcan/worktrees/`, `node_modules/`, `.next/`, 데이터베이스, 로컬 실행 캐시는 포함하지 않습니다.

Core 규칙이나 프로세스를 크게 바꾸는 변경은 구현 전에 이슈 또는 논의로 의도와 영향을 먼저 공유해 주세요.
