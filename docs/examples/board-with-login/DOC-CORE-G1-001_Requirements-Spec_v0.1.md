## 1. 문서 목적

본 문서는 로그인 기능이 있는 게시판 샘플 애플리케이션의 기능 요구사항, 비기능 요구사항, 인수기준, 범위를 정의한다.

## 2. 프로젝트 개요

| 항목 | 내용 |
| --- | --- |
| 프로젝트명 | Board With Login Sample |
| 목적 | 로그인 기반 게시판을 예제로 만들어 Vulcan-Anvil Ex 산출물 체계를 검증한다 |
| 대상 사용자 | 일반 사용자, 게시글 작성자 |
| 주요 업무/서비스 | 회원가입, 로그인, 게시글 조회, 게시글 작성/수정/삭제 |
| 적용 범위 | 단일 웹 애플리케이션 샘플 |
| 제외 범위 | 소셜 로그인, 관리자 기능, 댓글, 첨부파일, 검색 고도화 |
| 주요 제약사항 | 감리 대응 구조를 검증하기 위한 샘플이므로 기능 범위는 작게 유지한다 |

## 3. 용어 정의

| 용어 | 정의 | 비고 |
| --- | --- | --- |
| 사용자 | 게시판을 조회하거나 글을 작성하는 사람 | 비로그인/로그인 사용자 포함 |
| 작성자 | 게시글을 작성한 로그인 사용자 | 수정/삭제 권한 판단 기준 |
| 게시글 | 제목과 본문을 가진 게시판 콘텐츠 | 작성자와 작성일을 가진다 |
