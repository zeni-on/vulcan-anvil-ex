# Reference Standards

이 폴더는 프로젝트 생성 시 주입하거나 설계 단계에서 직접 참조할 표준 문서를 보관한다.

## 포함 문서

| 참조 코드 | 파일 | 적용 범위 |
| --- | --- | --- |
| `PUBLIC-DATA-STD-2025-11` | `공공데이터 공통표준(2025.11월).xlsx` | 용어, 단어, 도메인, 데이터 항목명 검토 |
| `KISA-SD-2021` | `소프트웨어_개발보안_가이드(2021.12.29).pdf` | 분석, 설계, 구현, 테스트 단계의 개발보안 기준 |
| `KISA-SWVD-2021` | `소프트웨어_보안약점_진단가이드(2021).pdf` | 보안약점 진단, 설계/구현 보안약점 점검 기준 |

## 사용 방식

- 요구사항, 기능명세, 프로그램명세, 화면설계, DB명세, 테스트 산출물에서 참조 ID를 남긴다.
- 데이터 표준 검토 시 공공데이터 공통표준 엑셀을 우선 대조한다.
- 보안 설계와 테스트 기준은 `KISA-SD-2021`을 우선 참조하고, 보안약점 진단 관점은 `KISA-SWVD-2021`로 보강한다.
