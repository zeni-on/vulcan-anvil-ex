# Reference Standards

이 폴더는 프로젝트 생성 시 주입하거나 설계 단계에서 직접 참조할 표준 문서를 보관한다.

## 포함 문서와 출처

| 참조 코드 | 파일 | 출처 | 적용 범위 |
| --- | --- | --- | --- |
| `PUBLIC-DATA-STD-2025-11` | `공공데이터 공통표준(2025.11월).xlsx` | 공공데이터포털, 행정안전부 공공데이터 공통표준 데이터. 이용허락범위: 제한 없음. | 용어, 단어, 도메인, 데이터 항목명 검토 |
| `KISA-SD-2021` | `소프트웨어_개발보안_가이드(2021.12.29).pdf` | 한국인터넷진흥원(KISA), 소프트웨어 개발 보안 가이드 게시 자료. | 분석, 설계, 구현, 테스트 단계의 개발보안 기준 |
| `KISA-SWVD-2021` | `소프트웨어_보안약점_진단가이드(2021).pdf` | 한국인터넷진흥원(KISA), 소프트웨어 보안약점 진단가이드 게시 자료. | 보안약점 진단, 설계/구현 보안약점 점검 기준 |

## 이용 조건 메모

- 이 폴더의 표준 원문은 Vulcan-Anvil Ex 산출물 작성 시 참조 기준으로 사용하기 위해 포함한다.
- 각 문서의 권리는 원 제공기관에 있으며, Vulcan-Anvil Ex의 Apache-2.0 라이선스는 이 원문 문서 자체에 적용되지 않는다.
- 원문을 재배포하거나 제출 산출물에 포함할 때는 제공기관의 최신 게시 페이지와 이용 조건을 다시 확인한다.

## 사용 방식

- 요구사항, 기능명세, 프로그램 설계, 화면설계, DB명세, 테스트 산출물에서 참조 ID를 남긴다.
- 데이터 표준 검토 시 공공데이터 공통표준 엑셀을 우선 대조한다.
- 보안 설계와 테스트 기준은 `KISA-SD-2021`을 우선 참조하고, 보안약점 진단 관점은 `KISA-SWVD-2021`로 보강한다.
